# 🚀 Task Manager MERN

A full-stack Task Manager application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

## Features

- User Registration
- User Login with JWT Authentication
- Protected Routes
- Create Tasks
- View Tasks
- Update Tasks
- Delete Tasks
- MongoDB Atlas Integration

## Tech Stack

### Frontend
- React
- React Router
- Axios
- Vite

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcryptjs
